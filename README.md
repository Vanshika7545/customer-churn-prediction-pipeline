# Customer Churn Prediction Pipeline

## 🚀 Overview

This project builds an end-to-end machine learning pipeline to predict customer churn and provide explainable insights into customer behavior.

## 🎯 Objectives

* Identify customers likely to churn
* Optimize for Recall to minimize missed churners
* Perform Cohort Analysis for business understanding
* Deploy an interactive web application

## 🛠️ Tech Stack

* Python (Pandas, NumPy)
* Scikit-learn, XGBoost
* SHAP (Explainable AI)
* FastAPI (Backend)
* Streamlit (Frontend)

## 📂 Project Structure

data/ → raw dataset
notebooks/ → analysis notebooks
src/ → ML pipeline code
app/ → backend API
models/ → saved models
